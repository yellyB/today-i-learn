# TIL

날짜: 2025년 11월 12일

lodash 의 디바운스 설정 중에 다음과 같은 옵션이 있다.

```python
debounce(fn, 500, { leading: true, trailing: false })
```

- `leading: true` → 첫 트리거시 즉시 전달된 함수 실행
- `trailing: false` → debounce의 기본 옵션은 true로, 마지막에 실행

⇒ 내가 원하는건 결제 버튼을 처음 클릭하면 재클릭 못하게 막는 기능이어서 위와 같은 옵션을 사용했다.
